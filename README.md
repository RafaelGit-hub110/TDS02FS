💻 Fundamentos de Programação
🎓 Apresentação da Disciplina

Bem-vindo ao universo da programação!

Esta disciplina foi desenvolvida para introduzir os conceitos essenciais da computação, capacitando o aluno a pensar de forma lógica, resolver problemas e construir soluções computacionais.

Aqui você dará os primeiros passos para se tornar um desenvolvedor, analista ou profissional da área de tecnologia.

🎯 Objetivo Geral

Desenvolver o raciocínio lógico e a capacidade de resolver problemas utilizando conceitos fundamentais de programação, programação orientada a objetos e estruturas de dados.

🧠 O que você vai aprender

✔ Pensar como um programador  
✔ Criar algoritmos  
✔ Tomar decisões usando lógica  
✔ Trabalhar com objetos e classes  
✔ Organizar dados de forma eficiente  
✔ Desenvolver soluções computacionais
